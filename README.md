# 🖊️ Mouse Drawing Canvas

A simple web canvas where users can draw using their mouse inside a bordered area.
This project allows freehand drawing and includes **Clear** and **Done** buttons to control the drawing.

---

## 🚀 Live Demo

[View Live Demo](https://your-netlify-link.netlify.app)

---

## ✨ Features

* Draw freely using your mouse
* Interactive canvas area
* **Clear button** to erase the drawing
* **Done button** to finish the drawing
* Simple and clean UI design
* Lightweight and fast

---

## 🛠️ Built With

* HTML
* CSS
* JavaScript
* Canvas API

## ▶️ How to Use

1. Open **index.html** in your browser
2. Move your mouse inside the canvas
3. Click and drag to start drawing
4. Press **Clear** to remove the drawing
5. Press **Done** when finished
